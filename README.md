# Smart Task & Timetable Planner

A mobile application developed using Flutter to help university students manage assignments, deadlines, revision sessions and personal commitments.

## Features

- Timetable Management
- Task Tracking
- Workload Planning
- Study Session Scheduling
- Local Data Storage
- Mobile-Friendly User Interface

## Technologies Used

- Flutter
- Dart
- Firebase
- SQLite

## Project Overview

This project was developed as a final-year software engineering project and received a First-Class grade. The application was designed to help students organise their academic workload through task management and timetable planning features.
